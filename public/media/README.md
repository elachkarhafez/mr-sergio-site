# Media Pipeline Notes

## Imported Instagram Assets

- Folder: `public/media/instagram`
- Generated by: `scripts/collect_instagram_media.mjs`
- Contains:
  - `images/` (post stills + reel posters)
  - `videos/` (downloaded reel clips)
  - `manifest.json` (post URL and asset mapping)
  - `links.json` (captured profile link list)

## Refresh commands

```bash
IG_CHROME_USER_DATA_DIR=.ig-session npm run media:import
```

If login is required:

```bash
IG_INTERACTIVE_LOGIN=1 IG_CHROME_USER_DATA_DIR=.ig-session npm run media:import
```

## Integration points

- `lib/site-data.ts`
  - `products[].images`
  - `lookbookFrames`
  - `instagramCinematic`

## Quality guidance

- Prefer sharp full-body suit imagery, tailored detail closeups, and clean store visuals.
- Keep top-priority homepage video assets concise for better mobile performance.
